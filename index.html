About

I build technical Unity assets with an emphasis on reliability, platform compatibility, and clear documentation. Primary areas of work include save systems, cloud sync integrations (UGS/Firebase), console authentication adapters, and editor tooling.

Relevant experience: systems design, plugin development, cross-platform testing, and packaging assets for distribution. All releases include clear offline documentation and integration examples.

Highlights

CrossSave Pro Prefab-free slot-based save system with AES encryption, UGS/Firebase cloud sync, console-ready auth adapters.
SaveSystem Debug UI Optional debug UI for runtime testing: save/load, slot switching, and mock cloud simulation.
Editor Utilities Tools for packaging, define-symbol management, and automated documentation export.
